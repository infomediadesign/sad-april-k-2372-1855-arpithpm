**Domigo** - Lodging facility for pets.
**Demo:** [http://domigo.netlify.com/](http://domigo.netlify.com/) - Frontend Deployed on [Netlify](http://domigo.netlify.com/), Backend deployed on Heroku.

You love your pet. You've also got a plan for the weekend which you cannot avoid. You are now looking for someone who can take care of your pet just like you do. Here comes Domigo. 

Domigo is a place where **Users** can signup, **Add**/**Remove** Pets to their accounts, **Make** a **booking** for their pet. 

There are 2 User Types:

 1. Customer User.
 2. Admin User.

**Customer User**
Customer User can make a booking, Add a pet, Remove a pet, Update the account details.

**Admin User**
Receive Payments, delete User Bookings, Modify User Bookings, Create Pricing Plans, Update & Modify Pricing Plans, Set Max Capacity for the accomodation, Set Admin User privileges.
